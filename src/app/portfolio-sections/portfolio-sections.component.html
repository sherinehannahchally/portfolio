  • Keystroke Decoding from Surface EMG Signals via Hybrid Deep Learning

    Python · PyTorch · CNN · BiLSTM · CTC Loss · sEMG

    Keystroke Decoding from Surface EMG Signals Report (PDF) GitHub repository

    • Implemented a CNN+BiLSTM hybrid encoder for decoding QWERTY keystrokes from surface electromyography (sEMG) signals using the emg2qwerty dataset, achieving the best test Character Error Rate (CER) of 14.96% among four evaluated architectures.
    • Architecture combined 3 stacked 1D temporal convolutions (256 channels, kernel size 3) for local feature extraction with 2 bidirectional LSTM layers (256 units per direction) for long-range sequence modeling, trained end-to-end with CTC loss.
    • Outperformed the TDS convolutional baseline (23.80% CER) and a plain bidirectional GRU (17.33% CER), demonstrating that combining local feature extraction with bidirectional recurrence is more effective than either mechanism alone.
